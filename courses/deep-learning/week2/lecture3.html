Detailed Notes

A loss function is a mathematical way to measure how wrong a model’s prediction is. It compares the output of the neural network with the true label or target value.

Different tasks use different loss functions. For example, regression problems often use Mean Squared Error, while classification problems often use Cross-Entropy Loss.

Loss functions are essential because they provide the signal that tells the model how to improve. During training, the network tries to reduce this loss over many iterations.
