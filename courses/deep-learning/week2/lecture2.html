Detailed Notes

Gradient descent is an optimization algorithm used to minimize the loss of a neural network. After gradients are calculated, the model updates its weights in the opposite direction of the gradient.

The size of each update is controlled by the learning rate. If the learning rate is too large, the model may overshoot the best solution. If it is too small, learning becomes very slow.

This method is central to deep learning because it allows the network to improve step by step across many iterations until it reaches better performance.
