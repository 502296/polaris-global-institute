Detailed Notes

A neural network is a computational structure made of connected units called neurons. These neurons are organized in layers. The input layer receives the raw data, hidden layers process that data, and the output layer produces the final prediction or result.

Each connection in the network has a weight, and each neuron may also have a bias. Together, these values determine how strongly information moves from one layer to another. During training, the model adjusts these values to improve performance.

This lecture is important because it builds the conceptual base for later topics such as activation functions, forward propagation, backpropagation, and optimization.

📘 Glossary
Neuron
A basic computational unit in a neural network.
Weight
A value that controls the strength of a connection between neurons.
Bias
An additional value that helps shift the output of a neuron.
